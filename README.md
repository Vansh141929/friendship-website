# FriendConnect

A friendly web application for video calls and instant messaging with friends.

## Access the App

Visit: **[friendtietb51in.github.io](https://friendtietb51in.github.io)**

The app is fully responsive and works on:
- 🖥️ Desktop
- 💻 Laptop
- 📱 Mobile phones
- 📲 Tablets

## Features

- **Video Call Preview** - Simulate video calls with friends
- **Instant Messaging** - Send and receive messages in real-time
- **Friend Management** - Add and manage your friends list
- **Friendly Prompts** - Pre-written conversation starters

## How to Use

1. Select a friend from the left sidebar
2. Type your message in the chat area
3. Click "Send message" to send it
4. Use the "Start Call" button to initiate a video call
5. Share friendly prompts to start conversations

## Technologies

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Responsive Design
